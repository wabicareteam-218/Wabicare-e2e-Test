# Working Docs — Agent Scratch Space

> This folder is for AI agents to create temporary markdown files.

## Purpose

AI agents (Cursor, Copilot, etc.) can freely create `.md` files here for:

- Thinking through complex problems
- Planning multi-step implementations  
- Drafting before finalizing
- Research notes and comparisons
- Debugging logs and analysis

## Rules

| ✅ Do | ❌ Don't |
|-------|---------|
| Create any `.md` files | Store final documentation |
| Use descriptive filenames | Put code files here |
| Keep work-in-progress notes | Leave stale files forever |

## When Done

Move finalized content to the appropriate location:

- **PRDs** → `docs/prd/`
- **Tasks** → `docs/tasks/`
- **Design specs** → `docs/design/`

Then delete the working doc.
